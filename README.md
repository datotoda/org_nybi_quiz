# quiz_nybi.py

This is [nybi.org](https://www.nybi.org/) quiz calculator. You can enter the quiz data and the functions below will return the answer

## quiz_1_api

#### function arguments

- Network ID

- Subnets Required

#### Return

- Subnet Mask

- 1st Available Host Address of Subnet 1

- Max # of hosts/subnet

## quiz_2_api

#### function argument

- IP Address

#### Return

- Network Address

- Broadcast Address

- Subnet Mask
